New Blog Project

1.  1) Название проекта: New Blog
    2) Предназначен для ведения личного блога по любой теме. Даёт возможность персонализированной настройки интерфейса (в будущем)

2.  1) минимальной версии необходимы 3 страницы: список постов; просмотр отдельного поста; редактирование поста; создание поста
    2) На странице "список постов" будут отображаться заголовки и часть текста последних 10 постов. 
    На странице "просмотр отдельного поста" будет отображаться заголовок и полный текст выбранного поста.
    На странице "редактирование поста" будет доступно редактирование заголовка и текста выбранного поста.
    На странице "создание поста" можно будет создать новый пост.
    3) Страницы будут свёрстаны с использованием стандартных компонентов bootstarap, кое-где по надобности изменены стили.
    4) Выше был указан минимальный функционал, поэтому он должен быть выполнен.
    5) В ближайшем будущем необходимо будет добавить авторизацию/регистрацию пользователей, которая будет реализована посредством использования технологии JWT (JSON Web Token)
    Также появится возможность оставлять реакции на статьи, личный кабинет пользователя, возможность выбора категории статьи для более удобного оформления блога 
    (вставка фотографий, кода, ...), сортировка контента по категориям.
